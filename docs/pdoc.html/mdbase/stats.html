mdbase.stats

Statistical calculations and plotting in MDBASE.

  • In MDBASE, statistical calculations and plotting are connected.
  • Current version of MDBASE calculates 4 different plot types + statistics:
    • Correlation plot + statistics: Pearson's r, p-values, R2 coefficients
    • Scatterplot matrix graph + statistics: Pearson's r + p-values
    • Scatterplot matrix tables (two heatmaps with Pearson's r + p-values')
    • Boxplots + statistics: p-values quantifying differences among groups

Important technical note - plot settings:

  • mdbase.stats set some default reasonable parameters for each plot.
  • In fact the program modifies global variable matplotlib.pyplot.rcParams.
  • The rcParams are modified in three steps:
    • The default rcParams are auto-pre-set in PlotParameters class.
    • In each plot initialization, the defaults are slightly auto-adjusted.
    • Moreover, in plot initialization, the user can modify my_rcParams arg.
  • The exception is ScatterplotMatrixGraph class:
    • The scatterplot matrix graph is somewhat specific plot type in seaborn.
    • The use sns-parameters is more reliable than the use of rcParams.
    • The user can modify selected sns params during the plot initialization.
  • DPI of the final image:
    • DPI=100 is auto-pre-set for all plots (for showing in Spyder/Jupyter).
    • DPI=300 is auto-pre-set (and can be changed) during the saving of a plot.

 118        '''
 119        Save a plot with a suplied name or a suitable default name.
 120
 121        Parameters
 122        ----------
 123        output_graph : str, optional, default is 'default'
 124            Name of the output graph, such as 'something.png'.
 125            If not given, some suitable default filename will be created.
 126            The output filename depends on the environment - see code below.
 127        dpi : int, optional, default is 300
 128            DPI resolution of the saved plot.
 129            
 130        Returns
 131        -------
 132        None

 142        # Get default output filename if it was not specified.
 143        if output_graph == 'default':
 144            calling_script = sys.argv[0]
 145            if calling_script.endswith('ipykernel_launcher.py'):
 146                # Calling script is a IPYNB-notebook (Jupyter),

 148                # => use a standard filename [my_plot_XXX.nb.png]
 149                # => this name will be created by an external function
 150                output_graph = PlotSaver.get_unique_filename()
 151            else:
 152                # Calling script is PY-script (CLI, Python, iPython),
 153                # the output finename will be [calling_script.py.png].
 154                output_graph = calling_script + '.png'
 155        
 156        # Save the plot (with DPI given by *dpi* argument).
 157        plt.savefig(output_graph, dpi=dpi)

 160    @staticmethod
 161    def get_unique_filename(basic_name='my_plot', ext1='.nb', ext2='.png'):
 162        '''
 163        Get some suitable default filename for an output graph.
 164        
 165        Parameters
 166        ----------
 167        basic_name : str, optional, default is 'my_plot'
 168            The basic name of the output graph.
 169        ext1 : str, optional, default is '.nb'
 170            The 1st extention of the output graph.
 171            The .nb value indicates that the output comes from a notebook.
 172        ext2 : str, optional, default is '.png'
 173            The 2nd extension of the output graph.
 174            The '.png' is used as a standard format.
 175
 176        Returns
 177        -------
 178        filename: str
 179            The name of the output graph.
 180            The output graph will be in current working direcrory.
 181            If the defaults are used, the output will be 'my_plot_XXX_nb.png',
 182            where XXX is the first number (001,002...)
 183            for which a file does not exist.
 184            
 185        Notes
 186        -----
 187        The default name 'my_plot_XXX_nb.png' has a reason.
 188        We assume that a default name has to be created if the graph was:
 189        (i) created is Jupyter (that is why '.nb') as the 1st extension and
 190        (ii) the user did not specify the name - that is my just (my_plot). 
 191        '''
 192        
 193        # Initialize counter
 194        i = 0
 195        
 196        # Test names {basic_name + counter + png}
 197        # until we find a file which does not exist.
 198        while True:
 199            i += 1
 200            filename = basic_name + f'_{i:03}' + ext1 + ext2
 201            if not(os.path.exists(filename)): break
 202        
 203        # Return the unique filename.
 204        return(filename)

 311    def regression(self, P1, P2, category=None, 
 312            rtype='linear', marker='k-', label=None,
 313            r_to_plot=None, p_to_plot=None, R2_to_plot=None):
 314        '''
 315        Add a regression curve to correlation plot
 316        (one plot can contain several regression curves).
 317
 318        Parameters
 319        ----------
 320        P1 : list or array
 321            X-values for regression.
 322        P2 : list or array
 323            Y-values for regression.
 324        category : bool, optional, the default is None
 325            Advanced option.
 326            The CorrelationPlot object (self)
 327            contains reference to data (self.df) = ref to pandas DataFrame.
 328            If category is given,
 329            only self.df[category] data are employed in regression.
 330            Example:
 331            To fit regression function only to TKR data,
 332            this function can be called as
 333            `CorrelationPlot.regression(P1,P2,category=(df.Itype=='THR')`.
 334        rtype : str, optional, default is 'linear'
 335            Regression type.
 336            Pre-defined function for regression.
 337            It can be one of: 'linear', 'linear_kx', 'quadratic', 'power'.
 338        marker : matplotlib format string, optional, the default is 'k-'
 339            Matplotlib format string defines the color and type of line/points.
 340            More details: GoogleSearch - matplotlib.pyplot.plot.
 341        label : str, optional, default is None
 342            Label that can be assigned to regression curve.
 343            If defined, this label will appear in the legend of the plot.
 344        r_to_plot : (float,float,int), optional, the default is None
 345            If r_to_plot contains list with three numbers (x,y,n),
 346            value of Pearson's r is added to graph/plot
 347            at position (x,y), rounded to (n) decimals.
 348        p_to_plot : (float,float,int), optional, the default is None
 349            If p_to_plot contains list with three numbers (x,y,n),
 350            p-value is added to graph/plot
 351            at position (x,y), rounded to (n) decimals.
 352        R2_to_plot : (float,float,int), optional, the default is None
 353            If R2_to_plot contains list with three numbers (x,y,n),
 354            R2 coefficient is added to graph/plot
 355            at position (x,y), rounded to (n) decimals.
 356            
 357        Returns
 358        -------
 359        None
 360            The output is the regression/fitting curve
 361            added to the *self.ax* object.
 362            Additional output are statistical coefficients
 363            printed to stdout.
 364            Yet additional (optional) output are statistical
 365            coefficients added to the plot (= to *self.ax* object).
 366        '''
 367        
 368        # Regression = fitting of pre-defined functions to XY-data.
 369        # Pre-defined function is selected according to rtype argument.
 370        # XY-data = P1,P2 = X-data, Y-data.
 371        # Statistical coefficients can be added to the graph as well.
 372        
 373        # Test, if category was given
 374        # (trick: pandas.Series cannot be compared with scalar => np.any
 375        if np.any(category) == None:
 376            ds = self.df
 377        else:
 378            ds = self.df[category]
 379        # Remove NaN values before regression
 380        # (scipy.optimize.curve_fit does not work with NaN's
 381        ds = ds[[P1,P2]]
 382        ds = ds.dropna()
 383        # Sort values
 384        ds = ds.sort_values(by=[P1,P2])
 385        # Get regression type & define regression function
 386        if rtype == 'linear':
 387            def linear(X,a,b): return(a*X + b)
 388            self.regression_func = linear
 389            self.regression_func_eq = 'y = a*x + b'
 390        elif rtype == 'linear_kx':
 391            def linear_kx(X,k): return(k*X)
 392            self.regression_func = linear_kx
 393            self.regression_func_eq = 'y = k*x'
 394        elif rtype == 'quadratic':
 395            def quadratic(X,a,b,c): return(a*X**2 + b*X + c)
 396            self.regression_func = quadratic
 397            self.regression_func_eq = 'y = ax**2 + b*x + c'
 398        elif rtype == 'power':
 399            def power(X,n,c): return(c*X**n)
 400            self.regression_func = power
 401            self.regression_func_eq = 'y = c * x**n'
 402        else:
 403            print('Unknown regression type (rtype) - no action.')    
 404        # Calculate regression
 405        X,Y = (ds[P1],ds[P2])
 406        par,cov = scipy.optimize.curve_fit(self.regression_func,X,Y)
 407        # Calculate Pearsons's coefficient r and p-value
 408        r_coeff,p_value = scipy.stats.pearsonr(X,Y)
 409        # Calculate coefficient of determination
 410        # R2 values: 1 = perfect, 0 = estimate~average(Y), negative = very bad 
 411        # https://en.wikipedia.org/wiki/Coefficient_of_determination
 412        Yave = np.average(Y)
 413        Yfit = self.regression_func(X,*par)
 414        SSres = np.sum((Y-Yfit)**2)
 415        SStot = np.sum((Y-Yave)**2)
 416        R2 = 1 - SSres/SStot
 417        # Show regression in graph
 418        self.ax.plot(X,self.regression_func(X,*par), marker, label=label)
 419        # Add statistical parameters to graph if required
 420        if r_to_plot != None:
 421            self._add_coeff_to_plot(r_coeff,r'$r$',*r_to_plot)
 422        if p_to_plot != None: 
 423            self._add_coeff_to_plot(p_value,r'$p$',*p_to_plot)
 424        if R2_to_plot != None: 
 425            self._add_coeff_to_plot(R2,r'$R^2$',*r_to_plot)
 426        # Print statistics to stdout
 427        self._print_statistics_to_stdout(par, r_coeff, p_value, R2)

 801    def _pairplot_statistics(self, r_coeffs, p_values,
 802                             r_pos, p_pos, r_decimals, p_decimals):
 803        '''
 804        Draw values of r-coefficients and p-values to the pairplot.
 805
 806        Parameters
 807        ----------
 808        r_coeffs : bool
 809            If True, Pearson correlation coefficients r are calculated.
 810            The Pearson r range from +1 (perfect positive correlation)
 811            through 0 (no correlation) to -1 (perfect negative correlation). 
 812        p_values : bool
 813            If True, p-values are calculated.
 814            In this case, the p-values represent the probability
 815            that we would get such a strong (or stronger) correlation
 816            just by coincidence.
 817        r_pos : (float,float)
 818            Coordinates for drawing the r coefficient in the graph.
 819        p_pos : (float,float)
 820            Coordinates for drawing the p-values in the graph.
 821        r_decimals : int
 822            The number of decimals for r coefficients.
 823        p_decimals : int
 824            The number of decimals for p-values.
 825
 826        Returns
 827        -------
 828        * None; the output are just the p-coeff and r-values in the pairplot.
 829        '''
 830        # Scipy + matplotlib: add Pearson's r to upper-left corners
 831        # Trick #0: for k in dict: for-cycle with dictionary returns keys
 832        # Trick #1: for i,k in enumerate(dict): trick0+enumerate => index+key
 833        # Trick #2: ax.text(x,y,s,transform=ax.transAxes) rel.coords ~ (0;1)
 834        # Trick #3: ax.text(*pos,...) => x,y coordinates in pos are unpacked
 835        # (*list and **dict = Python unpacking operators, for function calls
 836        for index1,column1 in enumerate(self.properties.keys()):
 837            for index2,column2 in enumerate(self.properties.keys()):
 838                (corr,pval) = scipy.stats.pearsonr(
 839                    self.df[column1],self.df[column2])
 840                if column1 != column2:
 841                    if r_coeffs == True:
 842                        self.grid.axes[index1,index2].text(
 843                            *r_pos, f'$r$ = {corr:.{r_decimals}f}',
 844                            transform=self.grid.axes[index1,index2].transAxes)
 845                    if p_values == True:
 846                        self.grid.axes[index1,index2].text(
 847                            *p_pos, f'$p$ = {pval:.{p_decimals}f}',
 848                            transform=self.grid.axes[index1,index2].transAxes)

 950        '''
 951        Draw correlation matrix tables for r-values and p-values.
 952
 953        Parameters
 954        ----------
 955        cmap_r : str, optional, default is 'Reds'
 956            Name of matplotlib.pyplot colormap.
 957            The colormap will be used for Pearson's r coefficients.
 958            List of available colormaps: `matplotlib.pyplot.colormaps()`.
 959        cmap_p : str, optional, default is 'Blues_r'

 962            List of available colormaps: `matplotlib.pyplot.colormaps()`.
 963        decimals_r : int, optional, default is 2
 964            Number of decimals in correlation matrix table showing r coeffs. 
 965        decimals_p : int, optional, default is 2
 966            Number of decimals in correlation matrix table showing p-values.
 967        cbars : bool, optional, default is True
 968            If True, the colormaps will contain side colorbars.

 975            They can be finalized (self.finalize) and saved (self.save) later.
 976        '''
 977        # (1) Prepare data for calculations
 978        ds = self.df[list(self.properties.keys())]
 979        ds = ds.dropna()
 980        # (2) Prepare empty tables to save results
 981        n = len(self.properties)
 982        r_values = np.zeros((n,n))
 983        p_values = np.zeros((n,n))
 984        # (3) Calculate correlations
 985        for (i,column1) in enumerate(self.properties.keys()):
 986            for (j,column2) in enumerate(self.properties.keys()):

 991        # (Flip rows so that the main diagonal started in upper left corner
 992        # (default: [0,0] = start of the main diagonal = lower left corner
 993        r_values = np.flipud(r_values)
 994        p_values = np.flipud(p_values)

 996        # ...draw cmatrix = heatmap

1002        # ...draw cmatrix = heatmap
1003        my_format = "."+str(decimals_p)+"f"

1006            linecolor='white', linewidth=2)
